1. Installation¶

Décompressez le fichier paquet-TableauSigne-x.x.7z Il contient:

  • le module tabsigne3.py
  • l’executable TabSigne.py
  • l’executable TabSigneSimplif.py
  • l’executable TabSigneGUI.py
  • une copie de la license GNU GPL v3
  • une copie du fichier tabvar.tex
  • une copie du fichier pdftabvar.tex
  • le dossier doc pour la documentation. Commencer par ouvrir index.html.

1.1. Logiciels PST+/PdfAdd¶

Ces scripts produisent des fichiers LaTeX et des fichiers xml lisibles dans PST+ et dans PdfAdd pour faire des tableaux de signe. Les outils PST+/PdfAdd permettent de modifier rapidement le tableau pour soigner le rendu final. On peut insérer la sortie dans un document, moyennant l’ajout du fichier tabvar.tex (pour PST+) ou pdftabvar.tex dans l’entête.

1.2. Python¶

Les scripts sont écrits en Python3 mais ils fonctionnent aussi avec Python2.7 (au moins les scripts en ligne de commande). Ils utilisent des modules standards ainsi que sympy et lxml; le script d’interface graphique TabSigneGUI.py utilise Qt5 qui produit de jolies fenêtres.

Attention, (pour Windows) lxml n’est installable que pour python <= 3.2. Globalement, pour Ububtu/Debian, il y a des paquets disponibles dans la distribution. Vous pourrez essayer:

# apt-get install python-sympy python-lxml python-qt5
# apt-get install python3-sympy python3-lxml python3-qt5

Ou de façon encore plus efficace, pour avoir les plus récentes versions des modules grâce à l’installateur python pip/pip3:

# pip install sympy
# pip3 install sympy

et de même pour lxml et PyQt5 (bien respecter la casse)
